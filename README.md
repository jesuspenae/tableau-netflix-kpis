# 🎬 Netflix KPIs Dashboard  
**Tableau · Data Visualization · KPI Analysis**

Dashboard interactivo que analiza el catálogo de Netflix desde una perspectiva estratégica: rendimiento del contenido, tendencias por categoría, distribución geográfica y métricas clave del catálogo.

---

## 📌 Business Case  
El objetivo del proyecto es ofrecer una visión clara y accionable del catálogo de Netflix, permitiendo identificar patrones en géneros, países, años de lanzamiento y otros atributos clave que influyen en la estrategia de contenido.
