# 🎬 Netflix KPIs Dashboard — Tableau

This project features an interactive Tableau dashboard designed to analyze key performance indicators (KPIs) across the Netflix catalog. The goal is to uncover insights related to content performance, genre trends, geographic distribution, and catalog evolution over time.

---

## 📊 Business Case

The objective of this analysis is to provide a clear, centralized view of Netflix’s global content library. By examining categories, genres, countries of origin, and release patterns, the dashboard helps identify strategic trends that influence content planning, catalog expansion, and audience engagement.

---

## 🚀 Technical Highlights

- **Data Preparation:** Cleaned and structured the Netflix dataset (CSV) to ensure consistency across fields such as country, genre, release year, and rating.
- **Tableau Modeling:** Built calculated fields and relationships to enable dynamic filtering by genre, country, rating, and content type.
- **Visual Design:** Designed a clean and intuitive interface with KPI cards, bar charts, maps, and trend lines to highlight the most relevant insights.
- **Interactivity:** Implemented filters and dashboard actions to allow users to explore the catalog from multiple analytical perspectives.

---

## 🔍 Key Insights

- **Genre Distribution:** Clear identification of the most dominant genres and categories within the platform.
- **Global Footprint:** Geographic visualization showing the countries contributing the most content to Netflix.
- **Catalog Evolution:** Trends in release years reveal how Netflix’s content library has expanded over time.
- **Content Mix:** Breakdown of Movies vs. TV Shows and how each category contributes to the overall catalog.
