# 🎬 Netflix KPIs Dashboard  
**Tableau · Data Visualization · KPI Analysis**

Un dashboard interactivo creado en Tableau para analizar el catálogo de Netflix desde una perspectiva estratégica: rendimiento del contenido, tendencias por categoría, distribución geográfica y métricas clave del catálogo.

---

## 📌 Business Case  
El objetivo del proyecto es ofrecer una visión clara y accionable del catálogo de Netflix, permitiendo identificar patrones en géneros, países, años de lanzamiento y otros atributos clave que influyen en la estrategia de contenido.

---

## 🚀 Key Features  
- **KPI Overview:** métricas clave del catálogo (títulos, géneros, países, ratings).  
- **Genre & Category Trends:** análisis visual de tendencias por tipo de contenido.  
- **Geographic Distribution:** mapa interactivo de países de origen.  
- **Release Year Patterns:** evolución del catálogo a lo largo del tiempo.  
- **Interactive Filters:** exploración dinámica por género, país, rating y más.

## 📂 Dataset  
El dataset incluye información pública sobre títulos de Netflix, con campos como:  
- Título  
- Tipo (Movie / TV Show)  
- Año de lanzamiento  
- País de origen  
- Género y categoría  
- Rating  
- Descripción  

El archivo CSV utilizado en este proyecto está incluido en este repositorio.
